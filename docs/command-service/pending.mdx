# Pending

<ApiMethod method="rpc" name="Pending" />

Get pending commands for host. Pending is means that their execution has not completed, be that successfully or unsuccessfully. This is defined as commands that do not have an exit_code. To mark a command as no longer pending, use the Update method and provide an exit_code.
