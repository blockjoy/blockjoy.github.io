# Refresh

<ApiMethod method="rpc" name="Refresh" />

Refresh API token.

<ApiMethod method="requests" name="AuthServiceRefreshRequest" />

<VersionType>blockjoy.v1.AuthServiceRefreshRequest</VersionType>

| Field                                                 | Type   | Description                                                                                                                                                                                                                                                                                                                                   |
| ----------------------------------------------------- | ------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| <TableCellWithOption text="refresh" info="optional"/> | string | This is the refresh token that was sent to you on login. It is optional here, because if it is not provided, the value from the metadata is used as a fallback. This means that if you are a browser client, you do not have to provide this field, as it was set by the call to AuthService/Login, or by the previous call to this endpoint. |
| token                                                 | string | This is your old token. It is allowed that this token is expired.                                                                                                                                                                                                                                                                             |

<ApiMethod method="returns" name="AuthServiceRefreshResponse" />

<VersionType>blockjoy.v1.AuthServiceRefreshResponse</VersionType>

| Field   | Type   | Description                                                                                                                                                 |
| ------- | ------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------- |
| rerfesh | string | The new refresh token. Just as with the call to AuthService/Login, this value is duplicated in the metadata, so a cookie can be set for the browser client. |
| token   | string | A newly minted Json Web Token.                                                                                                                              |
